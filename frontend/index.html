Web project example

Пример простого backend на django и frontend из HTML/CSS/JS для организации веб-приложения

  • 1) /1_imitating_fetch_and_one_fill/
    Пример заполнения карточки из JSON полученного от backend. При этом сам fetch() - имитация, данные подставляются прямо в JavaScript. Этот приём можно использовать если backend ещё не готов ответить на наш запрос.
  • 2) /2_real_fetch_and_template_gen/
    Пример заполнения страницы несколькими карточками из JSON полученного от backend. При этом сам fetch() уже проностью работает. Так же есть отпавка статьи на сервер без form, но с input'ами.
  • Доп.1) /page_fill.html
    Дополнительный пример, собирающий из JSON карточки по шаблону. Также демонстрирует реакцию стилей на ввод пользователя.
